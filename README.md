# skill-debugger-md

## Description
Step-through debugging for AI agent thought processes. Allows developers to set breakpoints inside SKILL.md files and inspect agent state.

## Usage
```bash
skill-debugger start --skill-path ./skills/example_skill.md
```

## Features
- Breakpoints in SKILL.md files
- Memory inspection
- Prompt context visualization
- Action plan inspection
